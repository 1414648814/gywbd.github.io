nodejs中的events模块和EventEmitter对象

事件是nodejs中的一个核心概念，node所声称的非阻塞IO就是通过事件驱动的模式实现的，而node使用js语言来编写程序，js本身就具有事件驱动的特性。在node中，基本上很多工作都是通过事件驱动完成的，处理用户请求、读写文件、读写数据库等都是等待某个事件触发后才开发工作。
